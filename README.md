## PROJECT INSTALLATION

- CLONE REPOSITORY

```
git clone https://github.com/nuronbeck/contacts_demo.git
```

```
cd contacts_demo
```

- INSTALL DEPENDENCIES

```
npm install
```

## USE 2 TERMINALS:

- RUN FAKE SERVER API

```
json-server db.json
```

- RUN CLIENT APP

```
npm run serve
```

## OPEN THE APP:

[http://localhost:8080/](http://localhost:8080/)
